Anatomy of a BUG Application - Building a Basic Application

Create a basic application for BUG that displays a message on the system output console.

  1. If necessary, switch perspective to the Dragonfly perspective.
  2. Select the New BUG Project icon in the toolbar.
  3. In the New BUG Project window, in the Name field, type: Hello World
  4. Click Finish.
  5. In the Project Explorer view, expand the node for the Hello World application.
  6. Expand the node for the hello_world package.
  7. Double-click on Activator.java to edit.
  8. Add System.out.println("Hello World :)"); after the line that begins with public void start.
  9. Select CTRL+S to save entry.
  10. To run this application, start or restart the Virtual BUG and look for the output in the Console.
