Overview

By extending the Device Management and Remote System Explorer plugins, this feature allows SDK users to utilize services on BUG such as ssh shell, remote file access, and OSGi bundle management.

To enable this feature, follow these steps:

  • 1. Open the Remote Systems Explorer perspective.
  • 2. Click the "Define a new connection" button on the "Remote Systems" view.
  • 3. In the wizard dialog, select "BUG Device" for remote system type.
  • 4. Enter the appropriate connection details and finish the wizard.
  • 5. Depending on what plugins are installed in your Eclipe environment, you may see different services available in your new connection. To interact with these services select and right-click to view the context menu.
