Debug an Application

To create a breakpoint in a BUG application and step through code:

  1. These steps assume you have created a Basic Application.
  2. In the Project Explorer view in Dragonfly, open the Hello World > hello_world node and double-click the Activator.java file to edit it.
  3. Locate the line of code that reads: System.out.println("Hello World :)"); Right-click on the line's left margin and select 'Toggle Breakpoint'. A blue dot appears in the margin to indicate an enabled breakpoint.
  4. In the Project Explorer view, right-click on the Hello World project and select Debug As > Virtual BUG. Eclipse displays a dialog box asking if you want to switch perspectives.
  5. Click Yes. In the Debug Perspective, the Java editor highlights the breakpointed line and stops the application.
  6. Press F8 to step through the code.
  7. Confirm that the application output displays in the Console view.
