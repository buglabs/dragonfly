Debug an Application

To create a breakpoint in a BUG application and step through code:

  1. In Dragonfly, in the Project Explorer, open the Hello World > helloworld node and double-click the Activator.java file to edit it.
  2. In the line of code: System.out.println("Hello World :)");
    Right-click on the line's left margin and select Toggle Breakpoint. A blue dot appears in margin to indicate a enabled breakpoint.
  3. In the Project Explorer, right-click on Hello World project and select Debug As > Virtual BUG. Eclipse displays a dialog box asking if you want to switch perspectives.
  4. Click Yes. In the Debug Perspective, the Java editor highlights the breakpointed line and stops the application.
  5. Press F8 to step through the code.
  6. Confirm that the application output in the Console view.
