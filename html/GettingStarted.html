Getting Started

To create, compile, and run a BUG application in C, follow these steps:

  • 1. Open the C/C++ perspective.
  • 2. Create a new C project, and select "BUG 2.0 SDK" as the project type.
  • 3. For configuration select "Pre-installed BUG 2.0 SDK". This configuration assumes that you have the Angstrom SDK already installed in its default location: /usr/local/angstrom.
  • 4. Finish the wizard, see the new project in you workbench.
  • 5. Create a source file, add some code, and select the "Build" button in the Eclipse toolbar.
  • 6. Now see the executable under the Binaries folder in the project. Copy this binary to your BUG and run it!
