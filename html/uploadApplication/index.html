Upload an Application to BUGnet

  1. In the Project Explorer, right-click the application and select Send to BUGnet. (Note: You may also Drag & Drop applications from Project Explorer into the BUGnet view)
  2. If necessary, log in with your BUGnet username and password.
  3. The BUGnet view will refresh and show your application.
  4. By Default the first time you upload an application it will be marked as Private on BUGnet. If you would like the Public to see it, you can change this setting by going to BUGnet and edit your application.
