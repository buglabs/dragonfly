Create a BUG Project

  1. If necessary, switch perspective to the Dragonfly perspective.
  2. Select the New BUG Project icon in the toolbar.
  3. In the Name field of the New BUG Project window, type: Hello World
  4. Click Finish, or you can click Next to add services for your application to consume.
  5. In the Project Explorer view, expand the node of your project.
  6. Expand the node for the package and create your Class files.
