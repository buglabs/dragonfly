Send Application to BUG

  1. In the Project Explorer, right-click the application and select Send to BUG.
  2. Turn your BUG on and connect it to your computer.
  3. Right click on the application in the Project Explorer pane.
  4. Select Send to BUG.
    1. One BUG Connection in My BUGs: The application is now available on it.
    2. Mulitple BUG Connections in My BUGs: you will have to select one from the BUG Connections pop up window and click OK. The application is now available on your choice.
