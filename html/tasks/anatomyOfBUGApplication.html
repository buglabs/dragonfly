Anatomy of a BUG Application - Building a Basic Application

Create a basic application for BUG that displays a message on the system output console.

  1. If necessary, switch Perspective to the Dragonfly perspective.
  2. Select the New BUG Project icon in the toolbar.
  3. In the New BUG Project window, in the Name field, type: Hello World .
  4. Click Finish.
  5. In left-side Project Explorer view, expand the node for the Hello World application.
  6. Expand the node for the helloworld package.
  7. Double-click on Activator.java to edit.
  8. In the Java editor, in the start method of activator class, type: System.out.println("Hello World :)"); so that it looks like this: 

    9. 

package helloworld;

import org.osgi.framework.BundleActivator;
import org.osgi.framework.BundleContext;

public class Activator implements BundleActivator {

	public void start(BundleContext context) throws Exception {
		// TODO Auto-generated method stub
		System.out.println("Hello World :)");
	}

	public void stop(BundleContext context) throws Exception {
		// TODO Auto-generated method stub
		
	}
 }

  9. Go to File > Save to save Activator.java.
