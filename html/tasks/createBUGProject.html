Create a BUG Project

  1. If necessary, switch Perspective to the Dragonfly perspective.
  2. Select the New BUG Project icon in the toolbar.
  3. In the New BUG Project window, in the Name field, type: Hello World
  4. Click Finish.
  5. In the Project Explorer view, expand the node by your project.
  6. Expand the node for the package.
  7. Double-click on Activator.java to edit.
  8. Enter your code in the file.
  9. Save the file.
